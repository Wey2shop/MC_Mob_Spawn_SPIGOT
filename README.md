# Spigot Plugin UW Commander

## Description
This Spigot plugin provides a set of useful functions and features to enhance your Minecraft server. 
It is designed to work with Spigot, and has been tested on MC 1.2.4

## Installation
To install this plugin on your Spigot server, follow these steps:

1. Download the latest version of the plugin JAR file from the releases page.
(https://github.com/Wey2shop/MC_Mob_Spawn_SPIGOT/releases)

2. Place the downloaded JAR file into the `plugins` folder of your Spigot server directory.

3. Start or restart your Spigot server. The plugin will be automatically loaded.

## Features
- **1:** Provides a custom command file UW located after first run in the plugins dir.
- **2:** Implements the ability to have long commands shortened
- **3:** Added some pre-defined customised entitys, Complete with Drops, Lore and Named weapons.
- **4:** Not limited to Custome entities Can be used for any other command in the game.

## Usage
Once the plugin is installed and loaded on your Spigot server, you can access its features in-game ny using /UW. Here are some examples:

-`/UW 1` summon the first entity saved in the UW/entities.yml file
-`/UW 2` summon the secound entity saved in the UW/entities.yml file
-`/UW 3` summon the third entity saved in the UW/entities.yml file
And so on.. 

## Configuration
The plugin provides a configuration file (`entities.yml`) this allows you to customize the abreviated commands and assign new entity variations.

To make changes to the configuration, follow these steps:

1. Stop your Spigot server.

2. Open the `Plugins/UW/entities.yml` file using a text editor.

3. Make the necessary modifications to the configuration options.

4. Save the file and start your Spigot server.
